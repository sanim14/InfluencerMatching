# Asset Linker / Influencer Match Platform

## Overview

Asset Linker is a web application that helps users discover and connect with influencers based on a product and target audience. By entering campaign details, the platform generates relevant influencer matches and suggested outreach messages, making it easier to initiate collaborations.

## Features

* Input product description and target audience
* Generate relevant influencer matches
* Personalized outreach message suggestions
* Simple and intuitive user interface

## How to Use

1. Enter a product description
2. Specify the target audience
3. Click "Find Matches"
4. View influencer recommendations and outreach messages

## Demo

Live app: https://asset-linker-sanikamisra1.replit.app/

## Installation

```bash
git clone https://github.com/sanim14/asset-linker.git
cd asset-linker
npm install
npm run dev
```
